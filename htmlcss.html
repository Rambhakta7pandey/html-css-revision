01 — HTML

Document Structure

Every HTML page follows a standard skeleton. The browser reads this to know what your page is and how to render it.

<!-- Document Type Declaration (always first!) --> <!DOCTYPE html> <html lang="en"> <!-- Root element --> <head> <!-- Metadata (not visible on page) --> <meta charset="UTF-8"/> <meta name="viewport" content="width=device-width, initial-scale=1.0"/> <title>Page Title</title> <link rel="stylesheet" href="style.css"/> </head> <body> <!-- Everything visible goes here --> <h1>Hello World</h1> <script src="script.js"></script> </body> </html>

Semantic HTML Tags

Semantic tags describe the meaning of content, not just its appearance.

<header> — Top of page / section <nav> — Navigation links <main> — Primary content <article> — Standalone content <section> — Grouped content <aside> — Side content <footer> — Bottom of page <figure> — Images with captions

Non-Semantic Tags

Used for layout/styling with no inherent meaning.

<div> — Block-level container <span> — Inline container // Difference: <div> starts on a new line (block) <span> stays inline with text // Use semantic tags when possible! // div/span = when nothing fits

04 — CSS

CSS Basics & Syntax

Three Ways to Add CSS

/* 1. External (Best practice) */ <link rel="stylesheet" href="style.css"/> /* 2. Internal (inside <head>) */ <style> h1 { color: red; } </style> /* 3. Inline (avoid if possible) */ <h1 style="color: red;">Hi</h1>

CSS Syntax

/* selector { property: value; } */ h1 { color: blue; /* text color */ font-size: 24px; /* size */ margin: 10px 20px; /* shorthand */ } /* Multiple selectors */ h1, h2, h3 { font-family: serif; }
💡 CSS Cascade = C in CSS. Styles "cascade" down from parent to child. The last rule wins when specificity is equal.
05 — CSS

The Box Model

Every HTML element is a rectangular box. The box model defines how that box is sized and spaced.

MARGIN (outside)
BORDER
PADDING (inside)
CONTENT
width × height
.element { /* CONTENT */ width: 300px; height: 200px; /* PADDING — space inside, between content and border */ padding: 20px; /* all 4 sides */ padding: 10px 20px; /* top/bottom left/right */ padding: 5px 10px 15px 20px; /* top right bottom left */ padding-top: 10px; /* BORDER — the visible edge */ border: 2px solid red; /* width style color */ border-radius: 8px; /* rounded corners */ /* MARGIN — space outside, between elements */ margin: 20px auto; /* auto = center horizontally */ /* CRITICAL: box-sizing */ box-sizing: border-box; /* padding+border INCLUDED in width */ box-sizing: content-box; /* default: width = content only */ }
💡 Always set * { box-sizing: border-box; } at the top of your CSS. It makes sizing much more predictable — what you set as width is the actual visible width.
06 — CSS

CSS Selectors

Basic Selectors

/* Element selector */ p { color: blue; } /* Class selector (. prefix) */ .my-class { color: red; } /* ID selector (# prefix) — unique per page */ #my-id { color: green; } /* Universal selector */ * { box-sizing: border-box; } /* Attribute selectors */ input[type="text"] { } /* exact match */ a[href^="https"] { } /* starts with */ a[href$=".pdf"] { } /* ends with */ a[href*="google"] { } /* contains */

Combinator Selectors

/* Descendant (space) — any level deep */ div p { } /* <p> inside <div> */ /* Child (>) — direct child only */ div > p { } /* <p> directly in <div> */ /* Adjacent sibling (+) */ h1 + p { } /* <p> right after <h1> */ /* General sibling (~) */ h1 ~ p { } /* all <p> after <h1> */ /* Multiple classes on same element */ .btn.primary { }

⚡ Specificity — Who wins?

When two rules conflict, the one with higher specificity wins.

/* Specificity score (points): */ * { } /* 0 points */ p { } /* 1 point (element) */ .class { } /* 10 points (class/pseudo-class/attribute) */ #id { } /* 100 points (id) */ style="" /* 1000 points (inline) */ color: red !important /* Overrides everything (avoid) */ /* Examples: */ div p { } /* 2 points */ .nav a { } /* 11 points */ #hero .title { } /* 110 points */

.container { display: grid; /* Define columns */ grid-template-columns: 200px 1fr 1fr; /* fixed + flexible */ grid-template-columns: repeat(3, 1fr); /* 3 equal cols */ grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* responsive! */ /* Define rows */ grid-template-rows: auto 1fr auto; /* Named areas */ grid-template-areas: "header header" "sidebar content" "footer footer"; gap: 16px; /* row-gap + column-gap */ justify-items: center; /* align items horizontally */ align-items: center; /* align items vertically */ } /* Children */ .header { grid-area: header; } .item { grid-column: span 2; /* span 2 columns */ grid-column: 1 / 3; /* from line 1 to 3 */ grid-row: span 2; /* span 2 rows */ }
💡 repeat(auto-fit, minmax(250px, 1fr)) is the most powerful CSS Grid trick — creates a fully responsive grid with NO media queries!

CSS Positioning

5 Position Values

static (default flow)
relative (offset from normal)
absolute (to parent)
fixed (to viewport)
/* static — default, no offset */ .el { position: static; } /* relative — offset from its normal spot */ .el { position: relative; top: 10px; left: 20px; } /* absolute — relative to nearest positioned parent */ .el { position: absolute; top: 0; right: 0; } /* fixed — relative to VIEWPORT, stays on scroll */ .navbar { position: fixed; top: 0; width: 100%; } /* sticky — relative until scrolled past */ .header { position: sticky; top: 0; }

Units Cheatsheet
px — pixels (absolute)
em — relative to parent's font-size
rem — relative to root font-size (best for accessibility)
% — relative to parent's size
vw — 1% of viewport width
vh — 1% of viewport height
clamp(min, val, max) — fluid sizing between limits

/* ::before — inserts content BEFORE the element */ .icon::before { content: '→ '; /* REQUIRED (even if empty: content: '') */ color: gold; font-weight: bold; } /* ::after — inserts content AFTER the element */ .verified::after { content: ' ✓'; color: green; } /* Common uses of ::before/::after */ .clearfix::after { /* classic float clear */ content: ''; display: block; clear: both; } .overlay::before { /* dark overlay on image */ content: ''; position: absolute; inset: 0; /* shorthand for top/right/bottom/left: 0 */ background: rgba(0,0,0,0.5); } /* Other pseudo-elements */ p::first-line { font-weight: bold; } p::first-letter { font-size: 3em; float: left; } /* drop cap */ ::selection { background: yellow; } /* highlighted text */ ::placeholder { color: gray; } /* input placeholder */
16 — CSS

CSS Custom Properties (Variables)

/* Define variables in :root (global scope) */ :root { --primary-color: #f5c542; --font-size-base: 16px; --spacing-md: 1.5rem; --border-radius: 8px; --shadow: 0 4px 20px rgba(0,0,0,0.3); } /* Use variables with var() */ .button { background: var(--primary-color); padding: var(--spacing-md); border-radius: var(--border-radius); box-shadow: var(--shadow); } /* With fallback value */ .card { color: var(--text-color, black); /* uses 'black' if --text-color undefined */ } /* Scope variables locally */ .dark-theme { --primary-color: #1a1a2e; /* overrides inside .dark-theme */ } /* Update with JavaScript */ document.documentElement.style.setProperty('--primary-color', 'red');
💡 CSS variables are live — change one variable and the entire page updates instantly. Perfect for themes, dark mode, and design systems.
Quick Reference

CSS Display Values

block

Takes full width, starts on new line. Default for: div, p, h1-h6, section

inline

Only as wide as content, stays in line. Default for: span, a, strong, em

inline-block

Inline flow but accepts width/height. Best for: buttons, tags

flex

Turns element into flex container. 1D layout system.

grid

Turns element into grid container. 2D layout system.

none

Completely removes from layout and page (not just hidden).

/* Visibility vs Display */ display: none; /* gone from layout, no space taken */ visibility: hidden; /* invisible but SPACE still taken */ opacity: 0; /* transparent but SPACE still taken */ /* Overflow */ overflow: visible; /* default, content spills out */ overflow: hidden; /* clips overflowing content */ overflow: scroll; /* always show scrollbar */ overflow: auto; /* scrollbar only when needed */ overflow-x: hidden; /* horizontal only */ /* Cursor */ cursor: pointer; /* hand cursor */ cursor: not-allowed; /* disabled */ cursor: grab; /* draggable */ /* Box shadow */ box-shadow: 0 4px 20px rgba(0,0,0,0.3); box-shadow: inset 0 2px 4px rgba(0,0,0,0.2); /* inner shadow */ box-shadow: 0 0 0 3px gold; /* outline trick */ /* Centering a div — THE classic problem */ /* Method 1: Flexbox */ .parent { display: flex; justify-content: center; align-items: center; } /* Method 2: Grid */ .parent { display: grid; place-items: center; } /* Method 3: Absolute + transform */ .child { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }
